# PulseTap

A simple but addictive reaction time game.

## How it works
- Wait for the screen to change
- Tap as fast as possible
- Your reaction time is measured in milliseconds
- Best score is saved locally

## Built with
- HTML
- CSS
- JavaScript

## Features
- Reaction time tracking
- Best score saved using localStorage
- Mobile friendly UI
